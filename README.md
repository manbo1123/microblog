# microblog
- DjangoのMTV理解のため、ブログアプリを作成しました。
- トップページに、投稿内容と投稿時刻をリスト表示(3件で分割表示)。
- ヘッダーに、トップページへのリンク、ログインページへのリンクを配置。

# 開発環境
- Dango2
- DB: PostgreSQL

# 機能
- ユーザー登録機能
- ブログ投稿・更新・削除機能
